# Student Applicant Dashboard

ALX Front-End Capstone Project – Role-Based Student / Applicant Management Dashboard

## Features
- Admin dashboard with status filtering
- Student dashboard
- Dynamic routing with React Router
- Role-based login system
- Clickable applicant detail view

## Tech Stack
- React
- React Router
- CSS Grid
- JavaScript

## How to Run

git clone https://github.com/QKayManL/student-applicant-dashboard.git
cd student-applicant-dashboard/frontend
npm install
npm run dev
